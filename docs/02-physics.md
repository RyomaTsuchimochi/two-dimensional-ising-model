# 物理的背景

## イジング模型とは

イジング模型は、磁性体の相転移を記述する最もシンプルな統計力学モデル。
各格子サイトにスピン s_i = ±1 を置き、隣接スピン間の相互作用を考える。

### ハミルトニアン

```
H = -J Σ_{<i,j>} s_i s_j
```

- J > 0：強磁性（同じ向きのスピンが低エネルギー）
- `<i,j>`：最近接ペアの和（重複なし）
- 本プロジェクトでは J = 1（デフォルト）

---

## 格子の設定

### 正方格子 L×L

- 全サイト数：N = L²
- 各サイトは上下左右の4近傍と相互作用
- 周期境界条件：端のサイトも4つの近傍を持つ

### インデックス対応

```
index = x + L * y    （x: 列, y: 行）
```

周期境界：
```
left  = x == 0   ? index + (L-1) : index - 1
right = x == L-1 ? index - (L-1) : index + 1
up    = y == 0   ? index + L*(L-1) : index - L
down  = y == L-1 ? index - L*(L-1) : index + L
```

## モンテカルロ法（メトロポリス法）

熱平衡状態でのアンサンブル平均を、マルコフ連鎖モンテカルロで計算する。

### 1回の更新試行

1. スピン i を選ぶ
2. 反転したときのエネルギー差 dE を計算
3. 受理確率 P に従いフリップを受理

```
dE = 2 * J * s_i * (s_left + s_right + s_up + s_down)

受理確率:
  dE <= 0 のとき: P = 1（必ず受理）
  dE > 0  のとき: P = exp(-β * dE)
```

### 1 MCS（モンテカルロステップ）

全 N = L² スピンに対して1回ずつ更新試行を行う。
本実装ではランダムな走査順（フィッシャー-イェーツシャッフル）を採用。

計算量：1 MCS あたり O(L²)

## 観測量

### エネルギーと磁化

| 量 | 定義 |
|---|---|
| エネルギー H | H = -J Σ s_i s_j（近接ペア） |
| エネルギー密度 e | e = H / N |
| 磁化 M | M = Σ s_i |
| 磁化密度 m | m = M / N |

### 応答関数（ゆらぎから計算）

| 量 | 定義 |
|---|---|
| 比熱 C | C = (β² / N)(⟨H²⟩ − ⟨H⟩²) |
| 磁化率 χ | χ = (β / N)(⟨M²⟩ − ⟨M⟩²) |

### Binder cumulant（U4）

```
U4 = 1 − ⟨M⁴⟩ / (3 ⟨M²⟩²)
```

- 低温（強磁性秩序）：U4 → 2/3
- 高温（常磁性無秩序）：U4 → 0
- **異なる L の U4(T) 曲線は転移温度 Tc でほぼ交わる**（有限サイズスケーリングの根拠）

---

## 有限サイズスケーリングと Tc 推定

### Binder 交点法

1. 複数のサイズ L1 < L2 で U4(T) を計算
2. U4_{L1}(T) と U4_{L2}(T) の交点温度 T_cross を求める
3. L が大きいほど T_cross は真の Tc に近づく

### 補助的確認：比熱ピーク

- 比熱 C(T) は Tc 付近でピークを持つ
- ピーク温度も L → ∞ で Tc に収束する傾向がある

### 厳密値

```
Tc = 2 / ln(1 + √2) ≈ 2.2692  (J = 1, k_B = 1)
```
