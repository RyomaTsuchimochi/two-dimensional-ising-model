Binder Cumulant U₄(T) — 有限サイズスケーリング

U₄ = 1 − ⟨M⁴⟩ / (3⟨M²⟩²)。低温（秩序相）で U₄ → 2/3、高温（無秩序相）で U₄ → 0。 異なる L の曲線が交わる温度が Tc の推定値。赤点線が厳密値 Tc = 2.2692。

比熱 C(T)

C = (β²/N)(⟨H²⟩ − ⟨H⟩²)。Tc 付近でピークを示し、L が大きいほど鋭くなる（発散に近づく）。

O(L²) スケーリング確認

横軸 L²、縦軸 1 MCS あたりの計算時間 (ms)。線形（ほぼ原点通過）であれば O(L²) を確認できる。 各スピン更新のコスト（ns/spin）は L に依存せず一定。
